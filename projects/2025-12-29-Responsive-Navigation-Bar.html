Welcome to My Website!

This is a demonstration of a responsive navigation bar crafted by a fresher student.

On larger screens (desktop/tablet), you'll see a horizontal navigation menu. When you resize your browser window to a smaller width (like on a mobile phone), the menu will transform into a "hamburger" icon. Click the icon to toggle the navigation menu.

This project uses HTML for structure, CSS for styling and responsiveness (including media queries and transitions), and a small amount of JavaScript to handle the mobile menu toggle.
